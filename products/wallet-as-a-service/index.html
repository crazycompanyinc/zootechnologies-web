FAQ

How is this different from Fireblocks or Circle?

Fireblocks and Circle are managed services with monthly fees and per-transaction costs. This is self-hosted — you own the infrastructure, you control the keys, and you pay zero ongoing fees. Ideal for startups and indie hackers.

Do I need blockchain experience?

Basic understanding of how blockchains work (wallets, transactions, gas) is helpful. The API abstracts away chain-specific complexity — our unified API works the same across all 12+ chains.

Is the code audited?

Yes. All wallet and transaction code has been security audited. Audit report is included. We follow industry best practices for key management, signing, and transaction handling.

Can I add custom chains?

Absolutely. The plugin architecture makes it straightforward to add new EVM-compatible chains in under a day. Non-EVM chains require more work but the adapter pattern is documented.

What about compliance?

The KYC/AML module integrates with all major providers. However, compliance is your responsibility — consult a legal professional for your jurisdiction. We provide the technical infrastructure, not legal advice.
